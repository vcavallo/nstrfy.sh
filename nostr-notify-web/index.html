    📖 About

    nstrfy is a nostr-based push notification system — like ntfy.sh but decentralized. Notifications travel as kind 7741 events through nostr relays.

    Fill out the form and hit Send. If you supply a recipient npub, the message is end-to-end encrypted (NIP-44). Without a recipient, it's a public broadcast to anyone listening on the topic.

    This page signs events with a shared demo identity. Listen mode and your-own-key signing are coming in later phases. See the repo.
